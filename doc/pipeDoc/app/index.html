Desenvolvimento Local


Principais características:


  • Aplicação independente do backend e de ambientes de staging e QA, permite desenvolver utilizando contratos e mocks locais
  • Live reload provendo refreshs automáticos no browser ao salvar arquivos
  • Desenvolvida com Angular, Bootstrap, Mustache e utiliza os Ciclos de Vida do Node.js

Repositório Principal do Projeto


  • Disponível no endereço: https://gituol.intranet/plataforma/sac/sac-templates.git
  • Entrega independente dos binários e do antigo sync do svn _templates com os ambientes Base e Staging.
  • Os commits na branch development iniciam o fluxo de integração contínua, gerenciado pelo Jenkins.
  • A branch Master deve ser evitada, pois apenas o Jenkins deve commitar nela após executar testes, controle de qualidade e garantir o build.

JsHint

  • Controle de qualidade da codificação, inicialmente adotadas as aderências:
  • browser: verifica se não foram usadas variáveis globais que conflitam com Objetos nativos da maioria dos browsers atuais, como 'document'
  • esnext: Aderência à sintaxe do ECMAScript 6
  • bitwise: Proíbe o uso de operadores simplificados, como & no lugar de &&
  • camelcase: Força o uso de CamelCase para métodos e UPPER_CASE para constantes
  • curly: Requer o uso de Chaves em Loopings e Condições, para melhorar e legibilidade
  • eqeqeq: Obriga o uso de deep equall, === no lugar de == em condições.
  • immed: Proíbe a chamada de funções sem parenteses para diferenciar métodos de objetos
  • indent: Checa se toda a identação está baseada em 4 espaços.
  • latedef: Proíbe o uso de variáveis que ainda não foram definidas.
  • newcap: Requer que objetos construtores sejam sempre nomeados como Classes, que são capitalizadas
  • noarg: Não permite utilizar arguments.caller, forma de chamar métodos retirada desde o ECMAScript 5
  • quotmark: Permite apenas o uso de aspas simples, para facilitar a adição de HTMl em strings, que geralmente tem aspas duplas
  • regexp: Evita regex inseguras com ^
  • undef: Proíbe o uso de variáveis indefinidas
  • unused: Não permite que variáveis sejam declaradas, porém nunca usadas.
  • strict: Força o encapsulamento do código com a definição inicial 'use strict';
  • trailing: Não permite que sejam deixados espaços em branco desnecessários
  • smarttabs: Enibe os avisos quando são usados tabs e espaços misturados para alinhamento.
  • globals: Põe o angular e o jquery como variáveis globais permitidas.

Camaleão / Mustache

Exemplo de uso notParsableRegex no setup.json, excluindo toda a pasta scripts

"notParsableRegex" : "(.*)(\\\\|\\/)scripts(\\\\|\\/)(.*)"


Exemplo de uso parsableRegex no setup.json, pegando toda a pasta scripts

"parsableRegex" : "(.*)(\\\\|\\/)scripts(\\\\|\\/)(.*)"


Exemplo de uso extensions no setup.json, pegando todas as extensões

"extensions" : "*"
  • O Camaleão foi evoluído para também copiar arquivos, sem passar pelo parser do Mustache previamente.

    Com isso viabilizou-se o desenvolvimento de Meta-Templates totalmente auto-contidos, característica importante para que possam ser distribuídas novas versões independentes e que não quebram versões anteriores e/ou de outros produtos.

    Foram Incluídos os novos filtros no Camaleão:
  • extensions - filtra, inclusivamente por extensões, os arquivos que serão gravados
  • parsableRegex - filtra arquivos e diretórios inclusivamente via Regex
  • notParsableRegex - filtra arquivos e diretórios exclusivamente via Regex

Branch Development

  • Durante o desenvolvimento, deve ser criada uma branch local que pode ser nomeada de acordo com sprints, histórias ou hotfixes, sempre com o prefixo sac-templates.
  • O merge com a branch desenvolvimento iniciará o Pipeline a seguir.
  • Atenção: para garantir a integridade do projeto, a branch Master que representa o ambiente de produção, só deve ser mergeada automaticamente pelo Jenkis no final do Pipeline.

  • Configura o node para baixar as dependências via http no lugar de https, para que o a máquina de Proxy do ambiente de integração refaça a conversão para https novamente depois.
